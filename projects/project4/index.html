RISC-V CNN Accelerator

DSD 期末專題，使用 Verilog / Vivado 實作 RISC-V 控制與 CNN 運算資料路徑。成果在 Final Project AT Ranking 中取得 Team 3 Rank 1。

Rank 1Final Project AT Ranking

System

Top module 為 RISCV_CNN，目標平台為 xc7a35tcsg324-1。設計包含 control path、memory interface 與 CNN datapath。

  • RISC-V control 連接 instruction / data memory。
  • CNN datapath 包含 unpacker、line buffer、MAC engine 與 quantizer。
  • 以 Vivado 2022.1 完成 synthesis 與驗證流程。
